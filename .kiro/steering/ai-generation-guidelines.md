## Overview
This steering document guides AI generation quality and domain-specific effectiveness across the GhostFrame framework. While the current examples focus on educational applications, these guidelines can be adapted for any domain built using the framework.

## Framework AI Generation Principles

### Domain-Agnostic Quality Standards
- Generated content must be directly derived from source material
- Avoid ambiguous or misleading information
- Provide clear explanations and context for all generated content
- Balance complexity levels appropriate to the target domain

### Extensibility Guidelines
- AI generation patterns should be adaptable to any domain
- Maintain consistent quality standards across different application types
- Enable domain-specific customization while preserving core framework behavior

## Educational Domain Demo Guidelines (Quiz Ghost)

### Question Quality Standards
- Questions must be directly derived from uploaded content
- Avoid ambiguous or trick questions
- Provide clear, educational explanations for all answers
- Balance difficulty levels within each quiz

### Question Types
- **Multiple Choice**: 4 options, 1 correct answer
- **True/False**: Clear statements with detailed explanations
- **Short Answer**: Open-ended questions for deeper thinking

## Narrative Domain Demo Guidelines (Story Spirit)

### Narrative Quality
- Weave source concepts naturally into engaging plots
- Maintain appropriate language and themes for target audience
- Include clear objectives within the narrative structure
- Balance entertainment with informational value

### Story Structure
- Clear beginning, middle, and end
- Source concepts as integral plot elements
- Memorable characters that embody key ideas
- Satisfying resolution that reinforces learning

## Flashcard Generation Guidelines

### Card Effectiveness
- Front: Clear, concise questions or prompts
- Back: Comprehensive but digestible answers
- Include memory aids and mnemonics when helpful
- Group related concepts for better retention

### Difficulty Progression
- Start with foundational concepts
- Build complexity gradually
- Include review cards for reinforcement
- Adapt to user performance patterns

## Content Processing Standards

### Text Extraction
- Preserve formatting and structure when possible
- Identify and extract key concepts automatically
- Maintain context and relationships between ideas
- Filter out irrelevant or duplicate information

### Topic Identification
- Use semantic analysis to identify main themes
- Create hierarchical topic structures
- Tag content with relevant educational categories
- Enable cross-referencing between related concepts
